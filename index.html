Project Overview

About This Project

This project simulates an autonomous self-healing river cleaning robot that navigates a 36×14 km river grid, scanning each cell for pollution using real sensor data loaded from a CSV dataset.

The robot classifies pollution into 7 types and 3 severity levels, then generates a detailed site-specific remediation plan for each affected cell including estimated cost, timeline, and step-by-step treatment actions.

MATLAB Data Visualisation Robotics Simulation Environmental Engineering CSV Dataset Autonomous Navigation
🗺️

Autonomous Navigation

Boustrophedon (snake) path planner with obstacle avoidance — rock outcrops, bridge pylons, debris jams.

📡

Real Sensor Data

Reads pH, dissolved oxygen, turbidity, and temperature from a 504-cell CSV dataset with status flags.

🧪

7 Pollution Types

Agricultural runoff, industrial discharge, plastic waste, sewage, oil spills, thermal pollution, mining acid drainage.

💊

Remediation Plans

Each polluted cell gets a site-specific, multi-step clean-up plan with cost estimates and compliance timelines.
